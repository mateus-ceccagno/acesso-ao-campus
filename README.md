<h1 align="center">Acesso ao campus API</h1>

## About

API para registro de entrada e saída de pessoas ao campus e setores com diferentes níveis de acesso

* REST
* Swagger 2
* Swagger UI

## Technologies

Core

* Java 11
* Spring Boot 2.5.3
* Maven 4.0

Outros

* Lombok
* Mapsctruct
* h2

## Run Project

Clone Project

```git
git clone 
```

API

```ssh
mvn spring-boot:run
```

SWAGGER URL
```
http://localhost:8080/swagger-ui/
```
